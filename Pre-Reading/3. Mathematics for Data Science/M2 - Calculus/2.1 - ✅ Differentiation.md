# First & Second Derivatives
## Intro
$\tfrac{dy}{dx}$ means the rate of change of $y$ with respect to $x$
- E.g., $y=x^2$  $\tfrac{dy}{dx} = 2x$
## First: $f'(x)$
Gradient function - dives the slope of the tangent line at any point
- $\tfrac{dy}{dx}$
## Second: $f''(x)$
The rate of change of the slope
- $\tfrac{d^2y}{dx^2}$
- If $f''(x) > 0$ -> curve bends upwards (minimum)
- If $f''(x) < 0$ -> curve bends downwards (maximum)
# Rules of Differentiation
## Rules
- Power rule: $\frac{d}{dx} x^n = nx^{n-1}$
- Product rule: $(uv)' = u'v + uv'$
- Quotient rule: $\left(\tfrac{u}{v}\right)' = \tfrac{u'v - uv'}{v^2}$
- Chain rule: $\text{If } y=f(g(x)), \text{ then } \frac{dy}{dx} = f'(g(x)) \cdot g'(x)$
## Formal limit definition
$$
f'(x) = \lim_{h \to 0} \tfrac{f(x+h) - f(x)}{h}
$$
The official definition of the derivative
### Parts
- $f(x)$ -> value of the function at $x$ (the $y$ value)
- $f(x+h)$ -> $y$ value of the point the smallest distance along from $x$ (as $h \to 0$)
- $f(x+h) - f(x)$ -> how much the function changed in the vertical direction
- $h$ -> how much $x$ increased
- $\tfrac{f(x+h) - f(x)}{h}$ -> the slope of the secant line
## Derivatives of Standard Functions
- Exponential: $\frac{d}{dx} e^{nx} = ne^{nx}$
- Natural log: $\frac{d}{dx} \ln(x) = \tfrac{1}{x}$
- General exponential: $\frac{d}{dx} a^x = a^x \ln(a)$
- Trig functions:
	- $\frac{d}{dx} \sin(x) = \cos(x)$  
	- $\frac{d}{dx} \cos(x) = -\sin(x)$  
	- $\frac{d}{dx} \tan(x) = \sec^2(x)$  
	- $\frac{d}{dx} \cot(x) = -\csc^2(x)$  
	- $\frac{d}{dx} \sec(x) = \sec(x)\tan(x)$  
	- $\frac{d}{dx} \csc(x) = -\csc(x)\cot(x)$
