# Meeting Agendas
This folder contains structured agendas prepared before academic and project-related meetings during the MSc.

Each file outlines the purpose of the meeting, key discussion topics, and intended outcomes, included to demonstrate **preparation, organisation, and documentation discipline**.

---

## Folder Purpose
- Maintain a clear record of planning before meetings with lecturers, advisors, or supervisors
- Provide quick reference points for follow-up actions or decisions made
- Evidence workflow maturity and communication structure within the wider academic portfolio

---

## Notes
- All agendas are written in Markdown for version control and readability
- No personal or sensitive content is included
- These files serve purely as examples of structured preparation and project management practice
